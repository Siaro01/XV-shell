XV shell

Bienvenido

XV shell es la herramienta definitiva diseñada para la comunidad hispanohablante. Optimiza tu flujo de trabajo y toma el control de tu entorno con comandos potentes y una sintaxis sencilla.

Guía de Instalación

Sigue estos pasos para tener XV shell funcionando en tu sistema:

  1. Abre tu terminal favorita.
  2. Ejecuta el siguiente comando para descargar el paquete: curl -sL https://xv-shell.com/install | bash
  3. Espera a que el proceso finalice automáticamente.
  4. Reinicia tu terminal o ejecuta source ~/.bashrc
